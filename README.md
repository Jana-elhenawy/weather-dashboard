# 🌦️ Weather Dashboard

An interactive weather dashboard built with HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

## Features
- Search weather by city name
- Current temperature, humidity, wind speed, UV index
- 5-day forecast cards
- Dynamic background and icons based on weather condition
- Metric / Imperial unit toggle
- Responsive design for mobile and desktop

## How to Run
1. Clone the repository
2. Get a free API key from [openweathermap.org](https://openweathermap.org/api)
3. Open `index.html` and paste your API key in the `API_KEY` variable at the top of `app.js`
4. Open `index.html` in any browser — no server required!

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- OpenWeatherMap API (free tier)
- CSS Grid & Flexbox
- CSS custom properties for theming
